# Loan Estimate Verification

## Introduction
This application is to verify the Loan Estimate document by comparing:
- Loan Estimate PDF generated by a Document Generator System
- Excel File with `label` and `value` generated by the Mortgage System.

## Prerequisites
- Python 3.11 or higher
- OpenAI API key

## Installation
1. Activate your virtual environment:

   ```powershell
   python -m venv venv
   source venv/bin/activate  # On Windows, use: venv\Scripts\activate
   ```
2. Install the app dependencies:

   ```powershell
   python.exe -m pip install -r LoanEstimateVerification\requirements.txt
   ```

3. Set your OpenAI API key (optional for fallback verification):

   ```powershell
   $env:OPENAI_API_KEY = "your_api_key"
   ```

4. Run the app:

   ```powershell
   python.exe LoanEstimateVerification\app.py
   ```

5. Visit `http://127.0.0.1:5000`

6. Sample Screen

## Notes

- The app uses direct PDF text matching first.
- If `OPENAI_API_KEY` is configured, it also uses OpenAI as a fallback when direct matching fails.
- The generated JSON file is saved to your system temporary directory and offered for download.
- There are Loan Estimate PDF and Excel file examples in `files` directory.
